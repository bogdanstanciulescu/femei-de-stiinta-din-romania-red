# femei-de-stiinta-din-romania-red
De-a lungul istoriei, femeile au avut un acces limitat la educație și cercetare, însă multe dintre ele au reușit să devină pioniere în domenii științifice importante. Resursa educationala digitala evidențiază aceste realizări și încurajează: egalitatea de șanse în educație interesul pentru știință și tehnologie.
